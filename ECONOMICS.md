# Unit Economics & Funnel Math

SpendLens is a free tool, so it generates revenue indirectly by feeding leads to Credex's core negotiation service.

## The Funnel (Monthly Projections)

1. **Top of Funnel**: 10,000 unique visitors to SpendLens.
2. **Audit Completion**: 2,000 audits run (20% conversion).
3. **High Savings Threshold (> $500/mo)**: 200 audits (10% of audits).
4. **Lead Capture (Email)**: 100 emails captured from high-savings users (50% conversion).
5. **Credex Client Conversion**: 10 new Credex clients signed (10% close rate).

## Unit Economics

- **Customer Acquisition Cost (CAC)**:
  - Traffic is organic/viral.
  - Claude API cost per audit: $0.015.
  - Total API cost for 2,000 audits: $30.
  - Cost per acquired Credex client: $3.00 ($30 / 10).

- **Lifetime Value (LTV)**:
  - Credex takes a 15% cut of the savings negotiated.
  - Average savings found: $1,000/mo ($12,000/yr).
  - Credex annual revenue per client: $1,800.
  - Assuming 2-year retention, LTV = $3,600.

- **LTV:CAC Ratio**: $3,600 : $3 = 1200:1 (Highly profitable lead gen).

## The Path to $1M ARR
To reach $1M ARR ($83,333/mo) for Credex via SpendLens:
- Need ~46 new clients per month.
- Requires ~460 high-savings leads/mo.
- Requires ~9,200 audits run/mo.
- Requires ~46,000 unique visitors/mo to SpendLens.
