# Changelog

All notable changes to SpendLens will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-05-24
### Added
- Core Next.js 14 application with Tailwind and Framer Motion.
- Deterministic Audit Engine for 8 AI tools (Cursor, Copilot, Claude, ChatGPT, Windsurf, Gemini, Anthropic API, OpenAI API).
- Pricing JSON data snapshot as of 2025-05-20.
- Supabase integration for audit storage and Lead capture.
- Anthropic Claude 3.5 Sonnet integration for AI summary generation.
- Resend email integration for delivering audit results.
- Public shareable URLs with PII-stripping logic.
- 12 comprehensive business and technical documentation files for Credex submission.
